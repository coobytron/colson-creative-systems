01Real-time ML2026

NCA Visual Systems

A neural cellular automata engine built for expressive, music-responsive image systems at 4K.

A working research platform connecting trained cellular behaviors, material studies, audio signals, and sequenced parameters. The system turns model behavior into an instrument an art director can perform with—not a one-click effect.

  • Metal

02WebGL experiment2026

RAW Signal

A live browser instrument inspired by corrupting RAW image data through audio-processing workflows.

The familiar Audacity databending process becomes immediate and reversible: image bytes are mapped to controllable visual signals, with the result updating while parameters move.

  • WebGL
  • GLSL
  • JavaScript
  • Signal mapping
03Generative interface2026

Mycelia

A living garden interface where language, pointer input, and simple rules become visible growth.

Text acts as seed data for an evolving cellular world. The project explores how a portfolio can behave like a system—launching immediately, hiding controls until needed, and making every visit slightly different.

  • p5.js
  • Cellular systems
  • Interaction
  • Web audio
04AI creative tool2025

Sound Entropy

A speech-responsive typography tool that makes syllable emphasis and language structure visible.

Built from a real communication need, the interface maps voice amplitude and timing into variable type. Words can be frozen, edited, and color-coded to support gestalt language processing without losing the play of live expression.

  • Speech input
  • Variable type
  • AAC
  • Prototyping

Approach

I use AI to build instruments for creative direction.

01

Make it controllable

The best generative systems expose meaningful decisions instead of hiding them behind a prompt.

02

Design in real time

Fast feedback makes technology feel like a material—something to push, tune, perform, and discover with.

03

Build from real needs

Tools become useful when technical possibility is shaped around a person, a workflow, or a communication problem.
