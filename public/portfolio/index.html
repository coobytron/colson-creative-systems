NCA Visual Systems

Real-time ML / Swift + Metal / 2026

+

A research platform that connects trained cellular behaviors, material studies, audio signals, and sequenced parameters. The model becomes an instrument an art director can perform with—not a one-click effect.

  • System design

02

RAW Signal

Databending instrument / WebGL / 2026

+

The familiar Audacity databending process becomes immediate and reversible: image bytes are mapped to controllable visual signals, with the result updating while parameters move.

  • Creative coding
  • WebGL
  • GLSL
  • Signal mapping
03

Mycelia

Living interface / WebGL + p5.js / 2026

+

Language and pointer input become seed data for an evolving cellular world. The interface launches directly into the garden and keeps its controls secondary to the experience.

  • Interaction design
  • Cellular systems
  • WebGL
  • Generative world
04

Sound Entropy

Speech-responsive type / AAC / 2025

+

Built from a real communication need, the interface maps voice amplitude and timing into variable type. Words can be frozen, edited, and color-coded to support gestalt language processing.

  • Product thinking
  • Speech input
  • Variable type
  • Accessibility
